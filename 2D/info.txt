Instruciones
************
Una vez ejecutado el binario se nos abrirá el menú principal con las siguientes opciones (seleccionamos con las
flechas y entramos con el ENTER/Espacio):
	-Play: Empezar a jugar
	-Password: Menu para introducir la contraseña
	-Exit: Cierra el juego

Una vez estemos jugando, manteniendo pulsado F2 muestra los controles, que son los siguientes:
	-Flechas para moverse
	-F1 para ver la password del nivel en el que estas
	-F2 para ver los controles

COMO PASARSE EL NIVEL:
**********************

Pisa todas las baldosas para que estan se activen, su color cambiara, una vez activadas todas saldra la llave que abrira el portal,+
para pasarte el nivel sal por el portal una vez que este se abra.
